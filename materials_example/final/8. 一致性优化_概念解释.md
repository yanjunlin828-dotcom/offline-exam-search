## 一致性优化问题的建模

### 从"所有节点共享 θ"到"引入局部副本 θᵢ"

**分布式优化的自然需求：** 将数据按行切分到 $N$ 个工作节点，每节点只有本地数据 $[X_i, y_i]$，看不到其他节点的数据。目标函数天然可分：

$$\min_\theta \sum_{i=1}^N f_i(\theta), \quad f_i(\theta) = \ell(X_i, y_i; \theta)$$

**矛盾：** $f_i$ 只依赖节点 $i$ 的数据，可以本地计算——但所有节点必须使用**同一个参数 $\theta$**，每步都需要通信。

**关键操作：** 给每个节点引入"局部副本" $\theta_i$，允许各节点暂时各自优化，最终通过约束强制达成一致：

$$\min_{\theta_1,\ldots,\theta_N} \sum_{i=1}^N f_i(\theta_i) \quad \text{但各节点仍可暂时不同}$$

### 一致性约束形式：min Σfᵢ(θᵢ), s.t. θᵢ = z

引入全局共识变量 $z$，把原问题改写为：

$$\min_{\theta_1,\ldots,\theta_N,z} \sum_{i=1}^N f_i(\theta_i) \quad \text{s.t.} \quad \theta_i = z, \quad i = 1, \ldots, N$$

| 变量 | 含义 | 维护方 |
|------|------|--------|
| $\theta_i$ | 节点 $i$ 的**局部参数副本** | 工作节点 $i$ |
| $z$ | 所有节点共同认可的**全局共识** | 主节点 |
| 约束 $\theta_i = z$ | 最终所有副本必须等于全局参数 | 约束强制 |

**为什么这个改写能纳入 ADMM：** 把所有 $\theta_i$ 堆成大向量 $x = (\theta_1,\ldots,\theta_N)$，目标函数 $= f(x) + g(z)$（其中 $g(z)=0$），约束 $\theta_i - z = 0$ 对应 ADMM 标准形式的 $Ax + Bz = c$——直接套用 ADMM 三步迭代。

### 等价性证明：改写后与原问题相同

约束 $\theta_i = z$ 强制所有副本等于同一个 $z$，代入目标函数：$\sum_i f_i(\theta_i) = \sum_i f_i(z)$，右边等于原来的 $\sum_i f_i(\theta)$（其中 $\theta = z$）。改写只是换了一种写法，问题本质不变。

## 一致性 ADMM 迭代公式推导

### 增广拉格朗日函数（含一致性约束）

每个节点 $i$ 有自己的对偶变量 $y_i$（缩放后记为 $u_i = y_i/\rho$），增广拉格朗日函数：

$$L_\rho = \sum_{i=1}^N f_i(\theta_i) + \sum_{i=1}^N y_i^T(\theta_i - z) + \frac{\rho}{2}\sum_{i=1}^N \|\theta_i - z\|_2^2$$

### θᵢ-update：各节点独立求解本地子问题

代入 ADMM 缩放形式（$A=I, B=-I, c=0$ 对每个节点）：

$$\boxed{\theta_i^{k+1} = \arg\min_{\theta_i} \left\{f_i(\theta_i) + \frac{\rho}{2}\|\theta_i - z^k + u_i^k\|_2^2\right\}}$$

**逐项解读：**
- $f_i(\theta_i)$：本地损失，只依赖节点 $i$ 的数据，**无需通信**
- $z^k$：上一轮主节点广播的全局参数（已知）
- $u_i^k$：节点 $i$ 的对偶变量（本地维护的历史偏差记录）
- $\frac{\rho}{2}\|\theta_i - z^k + u_i^k\|_2^2$：一致性惩罚，强迫 $\theta_i$ 靠近全局参数

**等价理解：** 等同于在本地损失上加以 $z^k - u_i^k$ 为中心的 L2 正则化：

$$\theta_i^{k+1} = \arg\min_{\theta_i}\left\{f_i(\theta_i) + \frac{\rho}{2}\left\|\theta_i - \underbrace{(z^k - u_i^k)}_{\text{锚点}}\right\|_2^2\right\}$$

**关键：** $N$ 个节点的 $\theta_i$ 更新**完全独立，可以完全并行**——每个节点只用本地数据和主节点广播的 $z^k$。

### z-update：全局参数更新为局部参数的平均值（z = mean(θᵢ + uᵢ)）

$z$ 更新需要解（$g(z) = 0$，只剩一致性惩罚）：

$$z^{k+1} = \arg\min_z \sum_{i=1}^N \frac{\rho}{2}\|\theta_i^{k+1} - z + u_i^k\|_2^2$$

**推导（为什么是均值）：** 对 $z$ 求导令其为零：

$$\sum_{i=1}^N \rho(z - \theta_i^{k+1} - u_i^k) = 0 \implies Nz = \sum_{i=1}^N(\theta_i^{k+1} + u_i^k)$$

$$\boxed{z^{k+1} = \frac{1}{N}\sum_{i=1}^N(\theta_i^{k+1} + u_i^k)}$$

**进一步化简：** 可以证明 $\bar{u}^k = \frac{1}{N}\sum_i u_i^k = 0$ 对所有轮次成立，所以：

$$z^{k+1} = \frac{1}{N}\sum_{i=1}^N \theta_i^{k+1} = \bar{\theta}^{k+1}$$

**全局参数 = 各节点局部参数的简单均值**（直觉：全局共识取所有节点的平均值）。

### uᵢ-update：对偶变量更新（uᵢ ← uᵢ + θᵢ - z）

$$\boxed{u_i^{k+1} = u_i^k + \theta_i^{k+1} - z^{k+1}}$$

$\theta_i - z$ 是本轮节点 $i$ 与全局参数的差距；$u_i$ 不断累积这个差距，充当"历史债务记账本"——下一轮 $\theta_i$ 更新时通过锚点 $z^k - u_i^k$ 逼迫偿还。

**并行性：** $u_i$ 更新只需要节点 $i$ 自己的 $\theta_i^{k+1}$ 和主节点广播的 $z^{k+1}$，各节点独立并行。

## 一致性 ADMM 通信模式

### 每轮通信内容：节点上报 θᵢ，主节点广播 z

**每轮完整通信序列：**

```
① 主节点广播 z^k（p 维向量）→ 所有工作节点

② 各工作节点并行（独立）：
   解本地子问题，得到 θᵢ^{k+1}
   发送 θᵢ^{k+1}（p 维向量）→ 主节点

③ 主节点汇总：
   z^{k+1} = 均值(θᵢ^{k+1})
   广播 z^{k+1} → 所有工作节点

④ 各工作节点（独立）：
   uᵢ^{k+1} = uᵢ^k + θᵢ^{k+1} - z^{k+1}
```

**每轮通信量：** $O(Np)$（$N$ 个节点各发送一个 $p$ 维向量），与数据量 $n$ **无关**。

### 与分布式 GD 的对比（通信量相同，但一致性 ADMM 可减少迭代次数）

| 对比维度 | 分布式 GD/SGD | 一致性 ADMM |
|---------|---|---|
| 各节点做什么 | 计算本地梯度（一次前向传播） | 解本地优化子问题（可能多次迭代） |
| 主节点做什么 | 聚合梯度并更新参数 | 对局部参数**求均值** |
| 通信内容 | 梯度（$p$ 维） | 参数（$p$ 维） |
| 每轮本地计算量 | 轻 | 重（解子问题） |
| 预期收敛轮数 | 多（每步贡献小） | 少（每步利用更多本地信息） |

**核心权衡：** ADMM 用更多本地计算换更少的通信轮次。**网络带宽是瓶颈时 ADMM 更优**；本地计算是瓶颈时 GD 更优。

## 带全局正则项的一致性 ADMM（Lec 14）

### 加入全局正则项 g(z) 的动机

Lec 13 解决的是纯损失求和 $\min \sum f_i(\theta)$。现实模型需要加**全局正则项**：

$$\min_x \left\{\sum_{i=1}^N f_i(x) + g(x)\right\}$$

$g(x)$ 是**全局**的（如 Lasso 的 $\lambda\|x\|_1$），不属于任何一个数据块，无法直接分到某个节点。

**解决策略：** 把正则项"推"到全局变量 $z$ 上——$\theta_i$ 负责各节点的本地损失，$z$ 同时负责正则化，约束 $\theta_i = z$ 维持一致性：

$$\min_{\theta_1,\ldots,\theta_N,z} \sum_{i=1}^N f_i(\theta_i) + g(z) \quad \text{s.t.} \quad \theta_i = z$$

与 Lec 13 的唯一区别：$g(z) \neq 0$，$z$ 更新不再是简单均值，而是近端算子。

### z-update 等于近端算子：prox_{g/Nρ}(均值)

$z$ 更新需要解（利用方差分解把 $N$ 个二次项合并）：

$$\min_z \left\{g(z) + \frac{N\rho}{2}\|z - \bar{\theta}^{k+1} - \bar{u}^k\|_2^2\right\}$$

**方差分解推导：** 令 $a_i = \theta_i^{k+1} + u_i^k$，$\bar{a} = \bar{\theta}^{k+1} + \bar{u}^k$，则：

$$\sum_{i=1}^N \|a_i - z\|^2 = \underbrace{\sum_{i=1}^N \|a_i - \bar{a}\|^2}_{\text{常数（与 }z\text{ 无关）}} + N\|\bar{a} - z\|^2$$

（交叉项 $\sum_i(a_i - \bar{a}) = 0$，消去）

丢掉常数项，$z$ 更新恰好是 $g$ 的近端算子：

$$\boxed{z^{k+1} = \text{prox}_{g/(N\rho)}\!\left(\bar{\theta}^{k+1} + \bar{u}^k\right)}$$

**直觉：** 主节点拿到各节点参数的均值，再通过 $g$ 的近端算子（如软阈值）"压缩"，产生全局稀疏性。

### 近端算子取代简单均值的数学推导

与 Lec 13 的 $z$ 更新对比：

| | Lec 13（无正则项）| Lec 14（有正则项）|
|--|---|---|
| $g(z)$ | $0$ | $\lambda\|z\|_1$（Lasso）|
| $z$ 更新 | $z = \bar{\theta}^{k+1}$（简单均值） | $z = S_{\lambda/(N\rho)}(\bar{\theta}^{k+1})$（均值 + 软阈值）|
| 物理含义 | 取平均就是全局共识 | 取平均后再压缩，引入稀疏性 |

## 分布式 Lasso 完整推导

### 分布式 Lasso 问题形式（Σ‖Xᵢθᵢ - yᵢ‖₂² + λ‖z‖₁，一致性约束 θᵢ = z）

将 Lasso 的数据按行切分到 $N$ 个节点：

$$\min_{x_1,\ldots,x_N,z} \sum_{i=1}^N \frac{1}{2}\|A_i x_i - b_i\|_2^2 + \lambda\|z\|_1 \quad \text{s.t.} \quad x_i = z$$

其中 $f_i(x_i) = \frac{1}{2}\|A_i x_i - b_i\|_2^2$（本地平方损失），$g(z) = \lambda\|z\|_1$（全局 L1 正则）。

### x-update：各节点求解带正则化的岭回归（闭式解）

对 $x_i$ 求导令其为零：

$$(A_i^T A_i + \rho I)x_i = A_i^T b_i + \rho(z^k - u_i^k)$$

$$\boxed{x_i^{k+1} = (A_i^T A_i + \rho I)^{-1}\!\left(A_i^T b_i + \rho(z^k - u_i^k)\right)}$$

**形式等同于岭回归：** 系数矩阵 $(A_i^T A_i + \rho I)$ 就是岭回归的系数矩阵（$\rho$ 扮演正则化参数角色），右端多了来自约束的"拉力" $\rho(z^k - u_i^k)$。

**两股拉力：** $A_i^T b_i$ 是数据拟合的拉力（往本地最优解方向），$\rho(z^k - u_i^k)$ 是约束的拉力（往全局共识方向）。

### z-update：全局 Soft-Thresholding（Lasso 的近端算子）

$g(z) = \lambda\|z\|_1$ 的近端算子是软阈值，代入 Lec 14 的 $z$ 更新公式：

$$\boxed{z^{k+1} = S_{\lambda/(N\rho)}\!\left(\bar{x}^{k+1} + \bar{u}^k\right)}$$

其中 $\bar{x}^{k+1} = \frac{1}{N}\sum_i x_i^{k+1}$，$S_\kappa(v) = \text{sign}(v)\cdot\max(|v|-\kappa,0)$。

**阈值为何是 $\lambda/(N\rho)$ 而非 $\lambda/\rho$：** 单机 ADMM 的 $z$ 更新二次项系数是 $\rho$；分布式时 $N$ 个约束合并，系数变为 $N\rho$，阈值因此缩小 $N$ 倍。

**直觉：** 主节点对各节点参数均值做软阈值操作，把均值中绝对值小的分量清零，产生全局稀疏解。

### u-update：各节点对偶变量更新

$$\boxed{u_i^{k+1} = u_i^k + x_i^{k+1} - z^{k+1}}$$

各节点并行独立更新，只需接收主节点广播的 $z^{k+1}$。

### Cholesky 预分解技巧（复杂度优化）

$(A_i^T A_i + \rho I)$ 在整个优化过程中**永远不变**（$A_i$ 是固定的本地数据，$\rho$ 是固定的超参数）。

**优化策略：**
- **初始化阶段（每个工作节点只做一次）：** Cholesky 分解 $(A_i^T A_i + \rho I)$，代价 $O(p^3)$
- **每轮迭代（重复 $T$ 次）：** 用已有分解做三角方程组回代，代价 $O(p^2)$

总计算节省：从 $T \times O(p^3)$ 降到 $O(p^3) + T \times O(p^2)$，当 $T$ 大时节省显著。

## 分布式 Lasso 停止条件

### 原始残差：θᵢ - z 的 RMS

$$\|r^k\|_2^2 = \sum_{i=1}^N \|x_i^k - z^k\|_2^2$$

衡量各节点局部参数与全局共识的差距之和（一致性约束的违反程度）。

**计算方式：** 各工作节点计算本地 $\|x_i^k - z^k\|^2$（标量），发给主节点求和取平方根。无需传完整向量，通信极省。

### 对偶残差：ρ(z - z_prev) 的 RMS

$$\|s^k\|_2 = \rho\sqrt{N} \|z^k - z^{k-1}\|_2$$

衡量全局参数 $z$ 的变化量（迭代是否已经稳定）。

**计算方式：** 主节点自己记录上一轮的 $z^{k-1}$，纯本地计算，无通信。

**停止条件：** $\|r^k\|_2 \leq \varepsilon_{\text{pri}}$ **且** $\|s^k\|_2 \leq \varepsilon_{\text{dual}}$，两个条件均满足才停止。

## 参数 ρ 的选择与影响

### ρ 过大时（强迫一致性，忽略数据拟合，收敛慢）

$\rho$ 大 → 一致性惩罚项 $\frac{\rho}{2}\|\theta_i - z\|^2$ 主导 → $\theta_i$ 被强迫紧贴 $z$，几乎无法根据本地数据优化 $f_i(\theta_i)$。

**效果：** 原始残差（$\|\theta_i - z\|$）收敛快，但数据拟合质量差，对偶残差（$z$ 的变化）收敛慢。

### ρ 过小时（重视数据拟合，一致性约束软弱，震荡）

$\rho$ 小 → $f_i(\theta_i)$ 主导 → 各节点各自优化本地损失，$\theta_i$ 分散，一致性约束形同虚设。

**效果：** 数据拟合好，但 $\theta_i$ 偏离 $z$ 太远，需要很多轮才能收敛，且路径震荡。

### 自适应 ρ 策略（根据残差比例动态调整）

监控两个残差的比值，动态调整 $\rho$：

```
if 原始残差 / 对偶残差 > 10:   ρ ← 2ρ   (原始残差大，增大 ρ 加强一致性约束)
if 对偶残差 / 原始残差 > 10:   ρ ← ρ/2  (对偶残差大，减小 ρ 放松一致性约束)
```

**注意：** 改变 $\rho$ 后必须重做 Cholesky 分解（因为 $(A_i^T A_i + \rho I)$ 变了）。

## 单机 ADMM vs 分布式（一致性）ADMM 对比

### 问题结构差异（单 min f(x)+g(z) vs min Σfᵢ(θᵢ)+g(z)）

| 维度 | 单机 ADMM（Lec 12）| 分布式一致性 ADMM（Lec 14）|
|------|---|---|
| 数据 | 全部数据在一台机器 | 数据切分到 $N$ 台机器 |
| 变量 | $x$（参数）和 $z$（副本） | $\theta_i$（各节点局部）和 $z$（全局） |
| 目标函数 | $f(x) + g(z)$ | $\sum_i f_i(\theta_i) + g(z)$ |
| $x/\theta_i$ 更新 | 单台机器解一个子问题 | $N$ 台机器并行解各自子问题 |
| $z$ 更新 | 近端算子（单个向量） | 近端算子（$N$ 个向量的均值） |

### 迭代角色差异（主节点协调 z，Worker 各自解本地子问题）

| 角色 | 单机 ADMM | 分布式一致性 ADMM |
|------|---|---|
| 主节点 | 不存在（单机） | 维护 $z$，计算均值，做软阈值，广播 $z$ |
| 工作节点 | 不存在（单机） | 维护 $\theta_i, u_i$，预存 Cholesky 分解，解本地岭回归 |

### Lasso 单机 ADMM vs 分布式 ADMM 公式对比

| 步骤 | 单机（Lec 12）| 分布式（Lec 14）|
|------|---|---|
| $x/\theta_i$ 更新 | $(X^TX+\rho I)^{-1}(X^Ty + \rho(z^k-u^k))$ | $(A_i^TA_i+\rho I)^{-1}(A_i^Tb_i + \rho(z^k-u_i^k))$ |
| 系数矩阵 | $X^TX + \rho I$（全部数据） | $A_i^TA_i + \rho I$（本地数据块，$N$ 个并行） |
| $z$ 更新输入 | $\beta^{k+1} + u^k$（单个向量） | $\bar{x}^{k+1} + \bar{u}^k$（均值向量） |
| $z$ 更新阈值 | $\lambda/\rho$ | $\lambda/(N\rho)$（多了 $N$ 分之一） |
| $u$ 更新 | $u \leftarrow u + \beta - z$ | $u_i \leftarrow u_i + x_i - z$（各节点独立） |

**最关键的区别：** 分布式版本的 $x_i$ 更新只用本地数据 $A_i, b_i$，因此可以在各节点完全并行；主节点只需收集 $p$ 维参数向量，计算均值并做软阈值，通信量始终为 $O(Np)$，与数据量 $n$ 无关。
